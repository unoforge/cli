# Commands

## Init
- `flexi-laravel init`
- `flexi-laravel init --new-laravel`
- `flexi-laravel init --no-flexiwind`
- `flexi-laravel init --js-path=resources/js --css-path=resources/css`

## Add
- `flexi-laravel add component-name`
- `flexi-laravel add component-a component-b --namespace=@flexiwind`
- `flexi-laravel add component-a --skip-deps`

## Icons
- `flexi-laravel fix-icons`

## Flux Cleanup
- `flexi-laravel clean:flux`
- `flexi-laravel clean:flux --force`
